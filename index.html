Cara Mendapatkan Google Client ID

Untuk menghubungkan aplikasi lokal ini ke Google Drive pribadi Anda, Anda perlu membuat Client ID OAuth2 di Google Cloud Console:

  1. Buka Google Cloud Console dan buat proyek baru.
  2. Cari dan aktifkan Google Drive API untuk proyek tersebut.
  3. Masuk ke menu APIs & Services > OAuth consent screen, pilih jenis User Type External, isi formulir consent screen (nama aplikasi, email) lalu simpan.
  4. Pada OAuth consent screen, pastikan Anda menambahkan email Google Anda ke daftar Test Users (karena proyek masih berstatus testing/draft).
  5. Masuk ke menu Credentials > Create Credentials > OAuth client ID.
  6. Pilih Application type: Web application.
  7. Pada bagian Authorized JavaScript origins, tambahkan URL lokal server Anda: http://localhost:8000
  8. Klik **Create** lalu salin string **Client ID** yang muncul dan tempelkan di kotak input pengaturan.

Catatan: Batasan otorisasi asal web mewajibkan Anda membuka aplikasi melalui alamat http://localhost:8000 agar autentikasi berjalan lancar.
